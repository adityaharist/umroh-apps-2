Aplikasi Umrah App adalah aplikasi untuk membantu anda yang akan melakukan perjalanan dengan kereta api. Dalam aplikasi ini anda dapat melihat jadwal kereta api untuk kelas Eksekutif, Bisnis, Ekonomi, dan Wisata dari stasiun awal keberangkatan sampai stasiun tujuan akhir. Selain itu, anda dapat melihat harga tiket atau tarif sesuai dengan kelas kereta api yang anda pilih.

Pada aplikasi ini, anda dapat memesan tiket kereta api melalui kanal pemesanan tiket yang telah tersedia, yaitu melalui layanan reservasi online yang bekerjasama dengan layanan Tiket.com.
